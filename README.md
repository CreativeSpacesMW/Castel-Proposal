## üöÄ How to Use

### Quick Start
1. Open `index.html` in any modern web browser
2. Click on any of the 4 cards to navigate to detailed pages
3. Use the "Back to Hub" button to return to the main page

### Editing the SLA Document
1. Navigate to the SLA page (sla.html)
2. Click any text to edit it inline
3. Click "Export as PDF" to download a clean, formatted PDF version
4. All edits are preserved until you refresh the page

### Customization
- All colors are defined as CSS variables in `styles.css`
- Edit content directly in the HTML files
- Fonts are loaded from Google Fonts (requires internet connection)
